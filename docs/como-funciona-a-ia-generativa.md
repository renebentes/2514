# Como funciona a IA Generativa?

- Large Language Model (LLM) - Modelo de Linguagem grande;
- É um tipo de programa de inteligência artificial treinado para **entender**, **gerar** e **manipular** **linguagem humana de forma natural** (NLP).

## Funcionamento

- Coleta de dados: LLMs são alimentadas com quantidade massiva de dados: livros, artigos, sites, redes sociais;
- Reconhecimento de padrões: durante o treinamento, aprendem a identificar relações entre as palavras;
- Previsão da próxima palavra: principal tarefa de uma LLM.

## Processo

- Ao receber um prompt (pergunta ou instrução feita em linguagem natural), a LLM decompõe em pequenas partes (tokens);
- Baseado no treinamento, é calculado a próxima palavra mais provável para seguir;
- Gera as respostas, palavra a palavra até formar frases inteiras.

## Considerações

- Não sabe o que diz;
- Não "lembra" de toda a conversa desde o início;
- A janela de contexto define o "pedaço" mais recente do texto (incluindo a instrução e respostas anteriores);
- Dados antigos tendem a ser os mais utilizados;
- O treinamento é mais genérico e menos específico.
